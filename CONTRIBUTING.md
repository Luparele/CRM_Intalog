# Contribuindo para o Zenith CRM

Obrigado por considerar contribuir para o Zenith CRM! 🎉

## Como Contribuir

### Reportando Bugs

Se você encontrou um bug, por favor abra uma [issue](https://github.com/Luparele/CRM_Intalog/issues) com:

- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs. atual
- Screenshots (se aplicável)
- Versão do Python e Django
- Sistema operacional

### Sugerindo Melhorias

Sugestões de novas funcionalidades são bem-vindas! Abra uma issue com:

- Descrição detalhada da funcionalidade
- Casos de uso
- Mockups ou exemplos (se aplicável)

### Pull Requests

1. **Fork o projeto**
```bash
git clone https://github.com/Luparele/CRM_Intalog.git
cd CRM_Intalog
```

2. **Crie uma branch para sua feature**
```bash
git checkout -b feature/minha-nova-feature
```

3. **Faça suas alterações**
   - Siga o estilo de código do projeto
   - Adicione testes se aplicável
   - Atualize a documentação

4. **Commit suas mudanças**
```bash
git commit -m "feat: adiciona nova funcionalidade X"
```

Padrões de commit:
- `feat:` Nova funcionalidade
- `fix:` Correção de bug
- `docs:` Documentação
- `style:` Formatação
- `refactor:` Refatoração
- `test:` Testes
- `chore:` Manutenção

5. **Push para sua branch**
```bash
git push origin feature/minha-nova-feature
```

6. **Abra um Pull Request**
